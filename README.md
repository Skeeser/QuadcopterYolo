# QuadcopterYolo
无人机的识别系统

<br>

## 功能特性
1. 通过瑞芯微的ai加速npu来实现硬件的加速
2. 实现识别跟踪功能


## 文件结构
- assets => 资源文件
- doc => 存放开发相关文档
- model => 存放模型
- include => 头文件
- cpp => cpp代码
- pythom => python代码
- README.md => 项目说明
